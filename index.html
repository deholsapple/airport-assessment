Data currency notice: Fields marked (auto) populate primarily from an embedded FAA 28-Day NASR dataset bundled with this app (dataset not loaded), with the live aviationweather.gov API and manual entry as fallbacks. This data goes stale on the FAA's 28-day publication cycle, and instrument-approach availability (RNAV/GPS approaches in particular) is not represented in this dataset at all — the app never infers "VFR only" from missing data. Always verify safety-critical fields against the current FAA Chart Supplement and NOTAMs before release. This tool does not replace dispatch/Chief Pilot verification.
